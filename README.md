## 📄 README: Currículo Digital Online

Olá! Seja bem-vindo(a) ao repositório do meu **Currículo Digital Online**!

Este projeto hospeda a versão mais atualizada e interativa do meu currículo profissional, acessível a qualquer momento pela web.

---

### 🚀 Sobre o Projeto

O objetivo principal deste projeto é fornecer uma plataforma moderna e de fácil acesso para visualizar minhas **qualificações, experiências, habilidades e formação acadêmica**.

Em vez de depender de documentos estáticos, este currículo digital oferece uma experiência aprimorada, permitindo que recrutadores e interessados naveguem pelas informações de forma organizada e visualmente agradável.

#### ✨ Principais Recursos

* **Design Responsivo e Moderno:** Layout adaptável a qualquer dispositivo (celular, tablet e desktop), construído sobre o Bootstrap 5.

* **Efeitos Visuais Dinâmicos:** Utilização de animações em scroll e transições suaves para introduzir elementos de forma dinâmica (via AOS e GLightbox).

* **Gestão de Projetos e Manuais:** Seção dedicada a projetos estratégicos de infraestrutura e uma central de manuais práticos para suporte preventivo.

* **Experiência Otimizada:** Conta com Pre-loader na abertura e Menu de Navegação Fixo para facilitar a exploração do perfil.

* **Recursos Interativos:** Efeito de Texto de Digitação (Typed.js) na introdução e Carrossel para Recomendações de colegas e parceiros (Swiper).

* **Proteção de Conteúdo:** Scripts em JavaScript para bloquear o clique com o botão direito e atalhos de inspeção (F12, Ctrl+U), protegendo a integridade das informações.

* **SEO Otimizado:** Configuração de tags Open Graph e Meta Tags para otimizar a indexação e a visualização em redes sociais como LinkedIn.

### 💻 Tecnologias Utilizadas

Este projeto foi construído com as seguintes tecnologias:

| Categoria | Tecnologia | Uso Principal |
| :--- | :--- | :--- |
| **Estrutura** | HTML5 | Conteúdo e Semântica |
| **Estilização** | CSS3, Bootstrap 5 | Layout, Estilo e Responsividade |
| **Interatividade** | JavaScript (ES6+) | Funções dinâmicas e manipulação do DOM |
| **Efeitos Visuais** | AOS.js, Typed.js | Animações de scroll e Efeito de Digitação. |
| **Componentes** | Swiper, GLightbox | Carrosséis de recomendações e visualização de mídia. |
| **Navegação** | Waypoints | Detecção de scroll para ativação de componentes. |
| **Hospedagem** | GitHub Pages | Hospedagem do projeto e disponobilização pública |
